# audio-reactive Template

Prototype template for **audio-reactive** applications.

## Status

**Active** — generated via `pnpm plantasonic create audio-reactive <slug>`.

## Source

Canonical template files live in:

```
packages/create-plantasonic-app/templates/audio-reactive/
```

## Generate

```bash
pnpm plantasonic create audio-reactive my-app
pnpm plantasonic create audio-reactive pulse-field --name "Pulse Field"
```

## Includes

- Pulse, Drift, and Glitch preset bundles with strong audio→visual mappings
- Audio-reactive inspector as default workspace panel
- Plugin that registers all reactive bundles

See `docs/GENERATOR_GUIDE.md` and the `create-prototype` skill.
