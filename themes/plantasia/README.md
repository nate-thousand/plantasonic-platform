# Plantasia Theme

Reusable Plantasia theme package placeholder.

## Status

`planned`

Plantasia currently uses the default Design System foundation and semantic theme behavior. No separate Plantasia token file is extracted in Phase 2.

## Boundary

- Do not rename existing Design System tokens.
- Do not change `packages/design-system/tokens/*.tokens.json` values.
- Do not change app output.

Future work may add Plantasia-specific semantic mappings here after they are identified as reusable platform theme definitions.
