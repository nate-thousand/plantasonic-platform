# {{APP_NAME}}

Audio-reactive platform consumer generated by `pnpm plantasonic create audio-reactive`.

## Run

```bash
pnpm install
pnpm dev
```

Opens at http://localhost:{{PORT}}.

## What this app owns

- Three reactive preset bundles (Pulse, Drift, Glitch) with tuned audio→visual mappings
- Bridge sensitivity/smoothing defaults per bundle
- App configuration and branding

## What this app does not own

- Design tokens, shell, or reusable UI (Design System)
- Sound/visual engine internals (platform adapters)
- Plugin runtime, persistence, MIDI routing (platform SDK)

## Customize

Edit `src/content/presetBundles.ts` to adjust mappings, sensitivity, and smoothing.
See `src/content/mappings.ts` for bridge reference values.

See [docs/GENERATOR_GUIDE.md](../../docs/GENERATOR_GUIDE.md) in the platform repo.
