# Release 1.0.0

First stable release of the Plantasonic Design System platform.

See [CHANGELOG.md](../CHANGELOG.md#100--2026-06-29).

## Highlights

- **Public API freeze** — 40 stable exports in `generated/api-surface.json`
- **Complete platform stack** — tokens, shell, components, motion, instrument, AI, prototype, ecosystem, studio
- **7 reference examples** — dashboard, instrument, creative-studio, generative-art, audio-reactive-installation, presentation, portfolio
- **163 automated tests** — full platform coverage
- **CLI** — `plantasonic create`, `spec`, `list` for apps and prototypes

## Upgrade from 1.0.0-rc.1

No breaking changes. Update version pin to `1.0.0`.

## Next milestone

Application development and ecosystem expansion — not new platform architecture.
