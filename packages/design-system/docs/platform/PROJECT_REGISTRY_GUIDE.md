# Project Registry Guide

Track every prototype in the ecosystem for cross-project maintenance and dependency management.

---

## Manifest schema

`platform.json` (generated by `createProject()`):

```json
{
  "id": "flower-study",
  "name": "Flower Study",
  "type": "generative-art",
  "version": "0.1.0",
  "engines": ["engine.visual"],
  "plugins": [],
  "assets": [],
  "workflows": ["workflow.import-assets", "workflow.generate-documentation"],
  "services": ["logging", "settings", "storage", "autosave"],
  "dependencies": { "plantasonic-design-system": "latest" },
  "documentation": ["README.md", "docs/PLATFORM.md"],
  "deployment": { "target": "preview", "status": "local" }
}
```

---

## Register projects

```typescript
import { createManifest, registerProject, projectRegistry, ProjectRegistry } from 'plantasonic-design-system/platform';

createManifest({
  id: 'bloom-room',
  name: 'Bloom Room',
  type: 'audio-reactive-installation',
  engines: ['engine.sound', 'engine.visual', 'engine.midi'],
});

// Isolated registry (tests, tooling)
const local = new ProjectRegistry();
local.register(manifest);
```

`createProject()` auto-registers to the global `projectRegistry`.

---

## Cross-project queries

```typescript
projectRegistry.byEngine('engine.visual');     // all visual projects
projectRegistry.byType('music-instrument');
projectRegistry.engineUsage();                 // Map<EngineId, projectId[]>
projectRegistry.byDeploymentStatus('production');
```

## AI context export

```typescript
import { buildProjectContext, exportProjectContext } from 'plantasonic-design-system/platform';

const ctx = buildProjectContext(manifest);
exportProjectContext(manifest, './docs/AI_CONTEXT.json');
```

Merged into `buildEcosystemContext()` for ecosystem-wide AI collaboration.
