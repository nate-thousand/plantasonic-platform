# __APP_NAME__

React + Bootstrap starter with the Plantasonic **Application Shell**.

## Includes

- Bootstrap component reference page
- Full design system theme layers
- Application Shell (`plantasonic-design-system/shell`) with navigation, routes, commands, docks, panels

## Commands

```bash
npm install
npm run dev
npm run build
```

## Customize

Edit `src/shell-config.ts` for navigation and shell regions. Page content lives in `src/pages/`.

See the design system showcase for full Bootstrap coverage: `npm run showcase:dev` in the design system repo.
